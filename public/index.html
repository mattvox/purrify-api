Purrify is a simple, easy-to-use RESTful service dedicated to delivering everything cat.

HTTP GET Requests

GET https://example.com/api/facts

Delivers to you a random cat fact from our database in standard JSON format.

QUERY PARAMETERS
Parameter Optional Description
Number Yes The number parameter determines how many amazing cat facts to return. 
GET https://example.com/api/facts/all

Delivers to you all cat facts from our database in standard JSON format.


HTTP POST Requests

POST https://example.com/api/facts

Purrify allows anyone to add cat facts to our database. Please note that our cat admins are very serious about the accuracy of newly submitted cat facts and each newly added fact will go through an extensive review process before becoming available through the API.

QUERY PARAMETERS
Parameter Optional Description
Source No The source parameter identifies the originating source of the cat fact.
Fact No The fact parameter contains the cat fact to be submitted for approval.
